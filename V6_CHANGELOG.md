# V6 Changelog

Doc Sprite Slicer Studio v6 is cumulative over v1-v5 and adds Animation Lab, Pose Lab, Sheet Remapper, Atlas Lab, runtime bundle export, onion-skin overlays, timeline QA, and V6 project schema fields.

## Added
- Timeline clips with per-frame duration metadata
- Onion-skin frame overlay on the main canvas
- Pose Lab for deterministic rig-part transform snapshots
- Sheet remap plan generator
- Atlas manifest preview/packer
- Runtime bundle JSON and ZIP exports
- Timeline validator integrated into QA
- V6 docs and sample project
