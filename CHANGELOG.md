# Changelog

## 6.0.0

V6 is the cumulative current version of Doc Sprite Slicer Studio.

### Added

- Animation Lab / Timeline mode.
- Onion-skin review controls.
- Pose Lab for deterministic rig-part transform checks.
- Sheet Remapper plan generation.
- Atlas Lab and runtime atlas manifest preview.
- Runtime bundle export target.
- Plugin registry foundation.
- Project migration normalization toward project format v6.
- Autosave/recovery package workflow.
- Asset pack importer workflow.
- Visual QA diff workflow.
- Batch logging/export package workflow.

### Repo hardening

- Added `package.json` with local scripts.
- Added static local server script.
- Added static smoke test.
- Added docs for project format, plugin system, QA rules, export profiles, and roadmap.
- Added sample v6 project file.
- Added release checklist.
- Added GitHub issue templates.
- Expanded ignore rules.
