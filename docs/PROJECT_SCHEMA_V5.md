# Project Schema V5

Required root fields:

- `type`
- `version`
- `projectFormatVersion`
- `source`
- `grid`
- `export`
- `qa`
- `parts`
- `pivots`
- `library`
- `recipe`
- `batch`
- `batchLogs`
- `profiles`
- `pluginsEnabled`
- `migrationReport`

V5 migrates older projects by filling missing fields with safe defaults.
