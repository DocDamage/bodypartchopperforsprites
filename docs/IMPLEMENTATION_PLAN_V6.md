# Implementation Plan V6

## Goal
Move from production-hardened slicing to animation/runtime prep without cluttering the UI.

## Modules
1. Animation Lab: timeline clips, durations, onion-skin review.
2. Pose Lab: deterministic pose snapshots for rig testing.
3. Sheet Remapper: layout conversion manifests.
4. Atlas Lab: runtime atlas manifest previews.
5. Runtime Bundle: engine-facing JSON package.
6. Timeline QA: validates clips before export.

## UI Rule
All new tools are in More modes and the Animation Lab menu. The canvas remains the center of the app.
