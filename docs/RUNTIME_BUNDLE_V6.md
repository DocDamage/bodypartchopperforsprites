# Runtime Bundle V6

The runtime bundle is a deterministic JSON package for engines and build pipelines.

It includes:
- Source metadata
- Grid and frame manifest
- Export profile
- Timeline clips
- Pivots and rig parts
- Atlas manifest
- Remap plan
- QA summary

This is not a full renderer. It is an engine-ready contract that Godot, Unity, RPG Maker helpers, KeterEngine, or custom tools can consume.
