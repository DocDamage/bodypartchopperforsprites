# V5 Implementation Plan

## Goal
Turn the V4 sprite workstation into a release-candidate foundation.

## Implemented

1. Plugin registry and Plugin Manager UI.
2. Project schema v5 and migration normalizer.
3. Autosave slots, backups, recovery package export.
4. Asset pack importer.
5. Visual QA diff validator.
6. Export preview generator.
7. Batch logs and failed-batch package export.
8. Plugin documentation and stub module folder layout.

## Deliberately deferred

- Cloud sync.
- AI auto-segmentation.
- Full drawing program.
- Full Spine replacement.
