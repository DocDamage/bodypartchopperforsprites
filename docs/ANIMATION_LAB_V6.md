# Animation Lab V6

Animation Lab creates named clips from row labels and frame cycles. Each clip records row, frame order, FPS, loop status, and per-frame duration data. Onion skin overlays neighboring frames over the selected cell so drift is visible without adding permanent panels to the canvas.

Recommended workflow:
1. Import sheet.
2. Set grid and row labels.
3. Open Timeline mode.
4. Add row clips.
5. Enable onion skin to inspect movement.
6. Export timeline JSON or runtime bundle.
