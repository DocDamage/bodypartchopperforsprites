# V5 Changelog

- Added internal plugin registry for importers, exporters, validators, batch steps, and migrations.
- Added Plugin Manager mode and Tools menu.
- Added project migration normalization for older `.spriteproject.json` files.
- Added autosave slots, recovery package export, and timestamped backups.
- Added asset-pack importer for PNG/WebP/JPEG assets plus optional JSON/CSV metadata.
- Added visual QA diff reports.
- Added batch logs and failed-batch export.
- Added export preview JSON.
- Added V5 project schema documentation and plugin guide.
